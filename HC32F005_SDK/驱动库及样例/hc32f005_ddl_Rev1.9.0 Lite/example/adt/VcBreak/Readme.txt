功能描述
================================================================================
本样例主要展示ADT VC中断刹车功能。
说明：
    

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32F005_STK

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）P23作为VC1的P输入，P24作为VC1的输出（仅用于示波器观察）
2）P23接GND
3）打开工程编译并运行
4）示波器观测P34(TIM4_CHA)和P35(TIM4_CHB)端口输出波形，开始运行时端口输出正常PWM波形，
   P24输出低
5）P23接VCC，此时P24输出高，PWM停止（按照刹车时预设电平输出）


================================================================================
注意
================================================================================
VC触发ADT刹车， 注意VC中断的时间，VC中断结束后刹车即失效，PWM恢复输出。
因此如有必要，需要在VC刹车中断内做PWM关断的处理。
